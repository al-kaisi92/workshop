# Workshop Technical Requirements

## Overview

This document outlines the software and setup requirements needed for students to participate in the Movie Review App coding workshop.

---

## Required Software

Please ensure the following software is installed on all student computers **before** the workshop:

### 1. Visual Studio Code (VS Code)

**Download:** https://code.visualstudio.com/

- Free code editor from Microsoft
- Available for Windows, macOS, and Linux
- Install the **Python extension** from the Extensions marketplace (search for "Python" by Microsoft)

### 2. Python 3.8 or Higher

**Download:** https://www.python.org/downloads/

- During installation on Windows, **tick the box "Add Python to PATH"** (this is important)
- To verify installation, open a terminal/command prompt and type:
  ```
  python --version
  ```
  or
  ```
  python3 --version
  ```
  It should display Python 3.8 or higher

### 3. Git (Optional but Recommended)

**Download:** https://git-scm.com/downloads

- Version control software
- Useful for cloning the workshop repository

### 4. GitHub Account (Required)

**Sign up:** https://github.com/join

- Each student needs their own GitHub account
- Students should create an account **before** the workshop day
- Free to create (use personal or school email)
- They will use this to access and store their project code

---

## Internet Access

Students will need internet access during the workshop for:

- Downloading Python packages (pip install)
- Viewing movie poster images from external URLs
- Accessing documentation if needed

---

## Hardware Requirements

- Any computer capable of running VS Code and Python
- Minimum 4GB RAM recommended
- At least 500MB free disk space

## Testing the Setup

To verify everything is working, students (or IT staff) can run these commands in a terminal:

```bash
# Check Python is installed
python --version

# Check pip is available
pip --version

# Test installing a package
pip install fastapi
```

If all commands work without errors, the setup is complete.
