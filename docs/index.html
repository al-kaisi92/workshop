> Getting Started

Setup Guide

Follow these steps to set up your development environment

1

Install Python

Windows

  1. Go to python.org/downloads
  2. Download Python 3.x.x
  3. Run installer - Check "Add Python to PATH"
  4. Verify: Open Command Prompt, type python --version

Mac

  1. Go to python.org/downloads
  2. Download Python 3.x.x
  3. Open the .pkg file and install
  4. Verify: Open Terminal, type python3 --version
2

Install VS Code

  1. Go to code.visualstudio.com
  2. Download for your OS
  3. Install and open VS Code
  4. Install Python extension (search "Python" in Extensions)
3

Create Virtual Environment

Windows

python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

Mac

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
4

Run the App

  1. Make sure (venv) is showing in terminal
  2. Run: uvicorn main:app --reload
  3. Open browser: http://127.0.0.1:8000

Create Web Service

  2. Connect your GitHub account
