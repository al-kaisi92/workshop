## About This Workshop

This workshop teaches complete beginners how to build an interactive movie review web application using **Python** and **FastAPI**. Students work through coding challenges to build features step-by-step.

**Duration:** 2 hours 45 minutes

**Tech Stack:** Python, FastAPI, Jinja2, JSON

## Getting Started

1. Download the starter files from `movie-review-app/`
2. Upload to Replit
3. Follow the challenges in `STUDENT_CHALLENGES.md`
4. Complete all 5 challenges to build the full app!

## What You'll Learn

- Python fundamentals (variables, functions, loops)
- Working with dictionaries and lists
- Reading JSON data files
- Building a web app with FastAPI
- HTML templates with Jinja2
