# Movie Review App Workshop

Welcome to the workshop materials for the **BTEC Student Movie Review App Workshop**.

## Quick Links

- [Student Challenges](./student-challenges)
- [Mentor Guide](./mentor-guide)

## About This Workshop

This workshop teaches complete beginners how to build an interactive movie review web application using HTML, CSS, and JavaScript. Students will deploy their finished apps to GitHub Pages.

**Duration:** 2 hours 45 minutes

## Getting Started

1. Download the starter files from `movie-review-app/`
2. Open `index.html` in your browser
3. Follow the challenges in `STUDENT_CHALLENGES.md`
4. Deploy to GitHub Pages when finished!
